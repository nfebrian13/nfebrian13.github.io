Inisialisasi projek di eclipse dan terhubung ke github Bagian 1.

1. Buat projek terlebih dahulu, contoh dibawah kita akan membuat project java

2. Maka akan terbentuk projek seperti berikut

3. Selanjutnya pilih windows preferences, lalu ketikan git dan pilih configuration.

4. Selanjutnya adalah mengisi credentials akun github kita seperti berikut.

5. Pilih projek yang telah kita buat, lalu klik kanan dan pilih team lalu share project seperti berikut.

6. Lalu checklist seperti langkah berikut, laluc klik pada bagian create repository. klik finish

7s. Setelah itu maka projek kita akan seperti berikut 'sample-git-project master', untuk inisialisasi projek terhubung ke github.
